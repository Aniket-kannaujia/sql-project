# 📚 SQL Project - Online Book Store

## 📌 Project Overview
This project demonstrates SQL-based data analysis using an **Online Book Store** dataset. It focuses on querying relational data to extract meaningful business insights through filtering, aggregation, joins, grouping, and ranking operations.

## 🛠️ Tools & Technologies
- SQL
- MySQL
- CSV Datasets
- Relational Database

## 📂 Dataset
The project consists of three related tables:
- 📚 Books
- 👥 Customers
- 🛒 Orders

These tables are connected using **Book_ID** and **Customer_ID** to perform relational analysis. :contentReference[oaicite:0]{index=0}
